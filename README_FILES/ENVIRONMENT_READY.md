## üéì Example Usage

### 1. Load Data
```bash

  -H "Content-Type: application/json" \
  -d '{"file_path": "ENROLLMENT.csv"}'
```

### 2. Get Courses
```bash
curl "http://localhost:8000/courses" | jq '.' | head -20
```


